How to install the Windows driver for Pocket SDR.

(1) Access the following link to get CyUSB (EZ-USB FX2LP) driver.

https://community.infineon.com/t5/Knowledge-Base-Articles/Drivers-for-EZ-USB-FX1-FX2LP-and-FX3-KBA94413/ta-p/248634

(2) Downlowd Drivers.zip and extract the file to an appropriate folder.

(3) Attach Pocket SDR RF frontend to your PC via USB cable.

(4) Open Windows Device Manager, select "EZ-USB" as "Universal Serial Bus Device".

(5) Select right-button menu Update Driver, select "Browse your computer for driver
software" and input the (2) folder path.

(6) After the driver installation, you find "Cypress FX2LP Sample Device" as
"Universal Serial Bus Controller" in Windows Device Manager.
